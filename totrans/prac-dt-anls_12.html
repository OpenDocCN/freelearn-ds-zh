Section 3: Working with Unstructured Big Data

Now that the foundations for data analysis have been established using structured data, in this section, the exciting frontier of unstructured social media and textual data will be explored. Here, we will learn about the various elements of supervised Natural Language Processing (NLP) and how to use a basic sentiment analysis model. Finally, we'll wrap up this book with a capstone project using open source data to provide insights.

This section includes the following chapters:

  • Chapter 10, Exploring Text Data and Unstructured Data
  • Chapter 11, Practical Sentiment Analysis
  • Chapter 12, Bringing It All Together
