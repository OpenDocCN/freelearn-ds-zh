# 前言

本书介绍了使用 Bash Shell 命令行的强大功能。Bash 是最广泛使用的 Shell，几乎在从烤面包机到高性能计算机的所有设备上都有使用。我们从基础开始，并在本书中迅速深入一些更高级的技能。

# 本书适合谁阅读

*命令行中的动手数据科学* 提供了使用命令行解决日常数据问题的有用技巧和窍门。本书面向的是那些几乎没有命令行经验，但有计算机科学背景和/或现代数据科学问题经验的读者。

你将学习如何在多个平台上设置命令行并根据你的喜好进行配置，如何查找命令的帮助，并学习如何创建可复用的脚本。你还将学习如何获取实际数据集，执行一些分析，并学习如何可视化数据。在书的最后，我们会介绍一些命令行的高级功能，以及接下来该如何深入学习。

此外，所有代码示例都可以在 Packt 的 GitHub 账户上下载。任何关于本书的更新将在 Packt 平台提供给你。

# 本书内容概览

第一章，*命令行中的数据科学与设置*，讲解如何在你选择的多个平台上安装和配置命令行。

第二章，*基础命令*，是一个关于如何使用命令行基础的动手演示，并讲解如何在需要时找到帮助。

第三章，*Shell 工作流、数据获取与处理*，真正开始进行一些基本的数据科学练习，使用实时数据集并根据需要自定义命令行环境。

第四章，*可复用的 Bash 脚本和在 Bash 中开发可复用代码*，基于前几章内容进行深入，介绍如何创建可复用的脚本和可视化内容。

第五章，*循环、函数和字符串处理*，是一个进阶的实战练习，讲解如何使用循环遍历数据并使用正则表达式进行探索。

第六章，*SQL、数学及总结*，是一个进阶的实战练习，帮助你将前几章学到的内容进行实际应用，我们还将介绍数据库、流处理和如何与 API 进行交互。
