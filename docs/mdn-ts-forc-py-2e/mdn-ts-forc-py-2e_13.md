# 第三部分

# 时间序列的深度学习

本部分聚焦于深度学习在解决时间序列问题中的应用。它从介绍必要的概念开始，逐步深入到适合处理时间序列数据的不同专门架构。同时，还讨论了深度学习中的全球模型以及一些提高其效果的策略。最后，我们深入探讨了生成概率预测，这是当今预测领域中的重要内容。

本部分包括以下章节：

+   *第十一章*，*深度学习简介*

+   *第十二章*，*时间序列深度学习的构建模块*

+   *第十三章*，*时间序列的常见建模模式*

+   *第十四章*，*时间序列的注意力机制与变换器*

+   *第十五章*，*全球深度学习预测模型的策略*

+   *第十六章*，*用于预测的专门深度学习架构*

+   *第十七章*，*概率预测及更多内容*
